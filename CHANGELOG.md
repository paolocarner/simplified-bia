# Changelog

All notable changes to the Business Impact Analysis Template will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-05

### Added
- Initial release of SMB Business Impact Analysis template
- Plain-language BIA framework with three-step process
- Simplified definitions for MTD, RPO, and RTO
- Step-by-step completion guide
- Common pitfalls section
- Practical examples throughout
- Implementation guidance for 2-3 hour completion timeframe

### Philosophy
- "Good enough beats perfect" approach
- Business-first, technology-second framework
- Focus on resource-constrained SMBs
- Designed to support both insurance decisions and compliance requirements

## Version Guidelines

**Major version (X.0.0)** - Significant structural changes that would require users to redo their BIAs
**Minor version (0.X.0)** - New sections, substantial additions, or improvements that don't break existing work
**Patch version (0.0.X)** - Clarifications, typo fixes, minor wording improvements
